Examen Git y Github

Alumn@: Diego Fernando Díaz Aristizábal

  1. Que es un sistema de control de versiones?

    Un sistema que te permite ver el historial de cambios que ha sufrido tu proyecto a lo largo de toda su existencia.

  2. Cual es la diferencia entre git y github?

    Git es el sistema como tal, la herramienta, mientras que gitub permite almacenar respositorios de GIT de forma centralizada.

  3. Cómo creo una rama en git?

    git checkout -b branch/name

  4. Cómo clono un repositorio?

    git clone URL
